# store-public-docs

Public document-store repository for files and materials intended to support public-facing sites or public document access.

## Repository type

Public document store.

## Purpose

This repository stores public documents and supporting files. It is intended to be legible to humans, GitHub, and connected tooling as a storage-oriented repository rather than an application-code repository.

## Public-safety boundary

This repository should contain public material only. Do not store private notes, credentials, unpublished intellectual property, internal handoff content, customer data, or secrets here.

## Operating notes

- Keep top-level context files simple and current.
- Use `REPO-MANIFEST.md` for repository-level metadata.
- Treat this as a public storage surface, not a private source-of-truth repository.
